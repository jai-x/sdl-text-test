# sdl-text-test

Reference program to verify the SDL Text Input event system

## requirements

* Make
* C compiler
* `SDL2`
* `SDL2_TTF`

## usage

Compile with make

Invoke with `./sdl-text-test <font.ttf>`

File `Silver.ttf` is included to use by default as it has decent Unicode
coverage and looks nice :)
